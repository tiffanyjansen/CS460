About the Assignment

For this assignment we were asked to put our Homework 8 Assignment onto the Cloud using Azure. The assignment wasn't too bad because I just followed all of his steps in the video found here: Video

Step 1: Create Account on Azure

This part wasn't so bad because I just followed the steps on their website to create an account. After creating it I just followed the video until I got it all done.

Step 2: Create Resource Group

This part was pretty explanatory because the website has a nice interface.

    On Azure Website:

  1. Click the Add Button
  2. Name the Group
  3. Click Create

Step 3: Create SQL Database

    On Azure Website:

  1. Go to SQL Database Tab
  2. Click Add
  3. Name It
  4. Put it in the Created Resource Group
  5. Keep the Source Blank
  6. **Create a New Server
  7. Select Basic Plan
  8. Click Create
**Creating a New Server Steps:
  1. Name it
  2. Give it a Username and Password
  3. Click Select

Step 4: Add a Firewall Rule

We add this so our local machine and our azure app can share the database that is in the cloud. (Opens the port for the IP address)

    On Azure Website:

  1. Go to Resource Group with Database
  2. Select the SQL Server in the Group
  3. Under Security Select Firewalls and virtual networks
  4. Name the Rule
  5. Copy and paste the IP Address into BOTH boxes
  6. Click Save at the top
  7. Wait for it to load

Step 5: Connect to DB from Local Machine

For this step I just used the SQL Server Object Explorer in Visual Studio.

    On Visual Studio:

  1. Go to SQL Server Object Explorer
  2. Select the Add SQL Server Thing Next to the Refresh Arrow
  3. Select Azure
  4. Find Server that you Added Eariler
  5. Type in Username and Password
  6. And BAM! Connected.
  7. Open the Up Script
  8. Click the Connect Icon
  9. Make sure the DB that you just Connected to is the one Connected
  10. Click the Green Execute Arrow
  11. Wait for it to load

Step 6: Get and Use the Connection String

This is important for accessing the database from the cloud.

    On Azure Website:

  1. Go to SQL Databases
  2. Select the Database we need the Connection String for
  3. Click the "Show database connection strings" Link
  4. Click the Click to Copy Icon
  5. Go to Visual Studio

    On Visual Studio:

  1. Open the Web.Config File
  2. Paste Connection String into the Connection String Section
  3. Put in Username and Password
  4. Make sure Names Match
  5. Make sure it Works.

Step 7: Create Web App on Azure

    On Azure Website:

  1. Go to App Services
  2. Click Add
  3. Select Web App
  4. Click Create
  5. Name It
  6. Use the Existing Resource Group
  7. Select a Free Plan
  8. Select Create

Step 8: Configure Connection String

    On Azure Website:

  1. Go to App Services
  2. Select Created App
  3. Go to Application Settings Tab
  4. Scroll To Connection Strings Section
  5. Type in Name Found in Context Class
  6. Paste Connection String from Web.Config into Value
  7. Select SQL Server instead of SQL Azure
  8. Click Save at the Top
  9. Wait for it to Load

Step 9: Publish to the Cloud

    On Visual Studio:

  1. Go to Build Tab
  2. Click on Publish
  3. Select "Select Existing"
  4. Click Publish
  5. Choose the Web App Created
  6. Click Ok
  7. DONE!
